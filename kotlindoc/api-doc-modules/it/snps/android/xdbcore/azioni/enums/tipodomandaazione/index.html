TipoDomandaAzione

enum class TipoDomandaAzione: Enum<TipoDomandaAzione>

Tipologia delle domande

EnumItems

Name Description
SI_NO

Tipologia domanda in cui sono presenti due possibili risposte:

  • SI: conferma l'operazione
  • NO: annulla l'operazione
INFO

Tipologia di domanda che si presenta qualora sia presente un problema nell'esecuzione della azione. Solitamente per qualche condizione di esecuzione non valida.
ATTENZIONE

Tipologia di domanda che si presenta qualora sia presente un valore (di tipo String) da fornire per il proseguimento della azione
DOMANDA_TESTO

Tipologia di domanda che si presenta qualora sia presente un valore (di tipo Numerico) da fornire per il proseguimento della azione
DOMANDA_VALORE

Tipologia di domanda che si presenta qualora siano presenti delle scelte, sottoforma di lista, da fornire per il proseguimento della azione
DOMANDA_SCELTA
