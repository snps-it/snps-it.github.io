Publishing

Le librerie raccolte in questo progetto sono pronte per essere pubblicate nel repository di Nexus di Sinapsi.

Per procedere alla pubblicazione di tali librerie bisogna che all'interno del progetto sia presente il file settings.properties contenente le informazioni necessarie per la pubblicazione e il download delle librerie dal repository Nexus

Il contenuto di tale file ï¿½ composto da coppie chiave-valore come da esempio:

nexusRepoReleases=[url repository di RELEASE]  
nexusRepoSnapshots=[url repository di SNAPSHOTS]
nexusUsername=[username per l'autenticazione al repository]
nexusPassword=[password per l'autenticazione al repository]

Download delle librerie da repository Nexus

Per poter effettuare il download delle librerie dal repository Nexus bisogna per prima cosa assicurarsi di aver inserito nella root del progetto il file settings.properties.

settings.properties

Il contenuto del file deve essere come segue:

nexusRepoPublic=[url repository public]
nexusUsername=[username per l'autenticazione al repository]
nexusPassword=[password per l'autenticazione al repository]

build.gradle.kts project level

All'interno del file build.gradle.kts bisogna andare ad indicare il repository Nexus dalla quale effettuare il download delle librerie pubblicate in precedenza:

// getting properties of Nexus repository
val fileProp = java.io.FileInputStream("$rootDir/settings.properties")
val prop = java.util.Properties()
prop.load(fileProp)

allprojects {
    repositories {
        ...
        maven {
            url = uri(prop.getProperty("nexusRepoPublic"))
            credentials {
                username = prop.getProperty("nexusUsername")
                password = prop.getProperty("nexusPassword")
            }

            authentication {
                create<BasicAuthentication>("basic")
            }
        }
    }
}

build.gradle.kts module level

All'interno di ogni build.gradle.kts presente nel modulo inserire nella sezione dependencies l'url della libreria da scaricare con la relativa versione, esempio: implementation("it.snps:[library]:[versione]")
