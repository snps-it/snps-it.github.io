Camera

Camera

Questa libreria permette l'avvio di un activity adibita per l'avvio della fotocamera customizzata.

Vengono gestite tre tipologie di funzioni per la fotocamera:

  • Semplice foto
  • Riconoscimento del testo
  • Lettura Barcode e QRCode

Installazione

Per Groovy

dependencies {
    implementation "it.snps:android-xdbcore-camera:<last-version>"
}

Per Kotlin DSL

dependencies {
    implementation("it.snps:android-xdbcore-camera:<last-version>")
}

Configurazione

La configurazione della libreria avviene allo stesso modo di come si vuole avviare un'altra activity in Android con il passaggio dell'intent contenente la tipologia di funzione che si vuole eseguire:

// Lanciare l'activity per semplice foto
binding.simplePhoto.setOnClickListener {
    startActivityForResult(Intent(this, CameraActivity::class.java).apply {
        putExtra(CameraActivity.EXTRA_CAMERA_FUNCTION, CameraFunction.SIMPLE_PHOTO)
    }, REQUEST_CODE)
}

// Lanciare l'activity per riconoscimento testo
binding.recognition.setOnClickListener {
    startActivityForResult(Intent(this, CameraActivity::class.java).apply {
        putExtra(CameraActivity.EXTRA_CAMERA_FUNCTION, CameraFunction.TEXT_RECOGNIZE)
    }, REQUEST_CODE)
}

// Lanciare l'activity per lettura barcode/qrcode
binding.barcode.setOnClickListener {
    startActivityForResult(Intent(this, CameraActivity::class.java).apply {
        putExtra(CameraActivity.EXTRA_CAMERA_FUNCTION, CameraFunction.BARCODE)
    }, REQUEST_CODE)
}

Al completamento della funzione richiesta viene richiamata l'activity chiamante tramite il setResult(...), bisogna necessario gestire il risultato proveniente dalla libreria.

Per le tipologie di funzioni disponibili consultare Camera Function

Gestione risultato

Per la gestione del risultato proveniente dalla libreria basta sovrascrivere il metodo onActivityResult().

La libreria aggiunge come intent di risposta i seguenti dati:

  • CameraActivity.EXTRA_BITMAP_PATH
  • CameraActivity.EXTRA_GENERIC_TEXT_RECOGNIZED
  • CameraActivity.EXTRA_MAIN_TEXT_RECOGNIZED

CameraActivity.EXTRA_BITMAP_PATH

Con questo dato viene fornito l'URI della foto scattata.

Tale dato viene fornito per tutte le funzioni ad eccetto della funzione CameraFunction.BARCODE in quanto con questa funzione non viene scattata nessuna foto ma vengono processati i vari frame fino a quando non risulta esserci un risultato nella lettura del Barcode/QRCode

CameraActivity.EXTRA_[GENERIC/MAIN]_TEXT_RECOGNITION

Con questo dato viene fornito il testo riconosciuto. Viene suddiviso in due categorie:

  • GENERIC
  • MAIN

Per GENERIC si intende tutto il testo presente nella foto scattato. Anche quello presente in MAIN.

Per MAIN si intente il testo presente all'interno del rettangolo presente nell'activity.

Nel caso in cui viene chiamata la libreria con la funzione CameraFunction.BARCODE viene restituito solamente il dato GENERIC
