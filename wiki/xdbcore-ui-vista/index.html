UI Vista

Visualizzazione record vista di Logica

Questa libreria permette di visualizzare una lista di record restituiti da una qualsiasi vista di Logica.

Installazione

Per Groovy

dependencies {
    implementation "it.snps:android-xdbcore-ui-vista:1.3.0"
    implementation "it.snps:android-common:1.0.5"
}

Per Kotlin DSL

dependencies {
    implementation("it.snps:android-xdbcore-ui-vista:1.3.0")
    implementation("it.snps:android-common:1.0.5")
}

Configurazione

La configurazione della libreria avviene tramite il VistaFragment.Builder che viene fornito al VistaListFragment che si occuperï¿½ di effettuare tutte le chiamate alle API di Logica per l'estrazione dei record della Vista fornita.

Esempio

val builder = VistaFragment.Builder<Vista>()
            // id della vista di Logica
            .withIdVista(1)
            // Classe che rappresenta un record
            .withModel(Vista::class.java)
            // Eventuale Header titolo
            .withHeaderTitle(R.string.title)
            // Map<String, String> da passare alla chiamata API (es: limit=-1)
            .withQueryMap(emptyMap())
           // Mostra un FAB con testo e drawable specificato in `Action`
            .withAction(Action(text = "Aggiungi",drawable = R.drawable.ic_baseline_add_24)) 
            // Mostra InputEditText per filtrare i risultati
            .withFilter()
            // Funzione per conversione oggetto restituito dalle API da fornire alla RecyclerView
            .withTransformationFunction {
                MapperContent(
                    id = it.ID,
                    caption = it.nome,
                    title = it.identificativo,
                    description = "Descrizione",
                    counter = it.ID.toString(),
                    imageUrl = "logo.jpg",
                    originalModel = it
                )
            }
            .build()

val vistaFragment = VistaListFragment.newInstance(builder.build())

supportFragmentManager.beginTransaction()
       .replace(binding.fragment.id, vistaFragment, VistaListFragment.TAG)
       .commit()

Interfacce

Tutte le interfacce devono essere implementate dall'activity o fragment che utlizzano il VistaListFragment qualora vogliano usufruire di alcune operazioni.

PagingLinster

Da utilizzare qualora si voglia utilizzare la Paging Library di Android

API Doc: PagingListener

    VistaFragment.Builder<Vista>()
        ...
        .withPaging()
        .build()

DaoListener

Da utilizzare qualora si voglia estrarre i dati solamente da Database e, in caso di refresh della lista, venga fatta una sincronizzazione.

API Doc: DaoListener

ActionListener

Da utilizzare qualora si voglia visualizzare un FAB per l'esecuzione di un'azione, ES: aggiunger un nuovo elemento alla lista

API Doc: ActionListener

    VistaFragment.Builder<Vista>()
        ...
        .withAction(Action(text = "Aggiungi",drawable = R.drawable.ic_baseline_add_24))
        .build()

ClickItemListener

Da utilizzare qualora si voglia eseguire qualche operazione al click di un elemento della lista

API Doc: ClickItemListener

LongClickItemListener

Da utilizzare qualora si voglia gestire il LongClick sull'elemento della lista

API Doc: LongClickItemListener
