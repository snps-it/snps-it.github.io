Azioni

Esecuzione azioni di Logica

Questa libreria permette l'esecuzione delle azioni di Logica da un APP Android.

Vengono gestiti tutti i tipi di domanda attualmente disponibili nel back-end di Logica.

API Doc: Tipo Domande Azioni

Installazione

Per Groovy

dependencies {
    implementation "it.snps:android-xdbcore-azioni:<last-version>"
}

Per Kotlin DSL

dependencies {
    implementation("it.snps:android-xdbcore-azioni:<last-version>")
}

Configurazione

La configurazione della libreria avviene tramite il EseguiAzione.Builder che viene fornito al DialogFragment che si occupa dell'esecuzione dell'azione.

Esempio

val builder = EseguiAzione.Builder()
            // id dell'azione da eseguire
            .withAzione("ID_AZIONE")
            // lista di id record sulla quale eseguire la azione
            .withRecords(listOf(2316))
            // eventuali azioni da eseguire a seguito di un'operazione su entitï¿½ di Logica
            .withAzioni(listOf(azioni))
            .build()

        EsecutoreAzioni.newInstance(builder)
            .show(supportFragmentManager, EsecutoreAzioni.TAG)

Interfacce

Tutte le interfacce devono essere implementate dall'activity o fragment che utilizza tale libreria

EsecutoreAzioneListener

interface EsecutoreAzioniListener {
        fun onAzioneComplete(result: ResultWrapper<XdbCoreAzionePost>)
    }

Viene chiamato il metodo onAzioneComplete al termine dell'esecuzione della azione con il body della risposta
